# 待办事项应用

一个简洁优雅的待办事项管理应用，采用黑白极简设计风格。

## 功能特性

- ✅ 添加、删除待办事项
- ✅ 标记任务完成状态
- ✅ 任务筛选（全部/进行中/已完成）
- ✅ 实时统计显示
- 🎨 黑白极简设计风格
- 📱 响应式设计，支持移动端

## 技术栈

- **前端框架**: Next.js 16 + React 19
- **开发语言**: TypeScript
- **样式框架**: Tailwind CSS
- **构建工具**: Next.js 内置

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

应用将在 [http://localhost:3000](http://localhost:3000) 上运行。

### 构建生产版本

```bash
npm run build
npm start
```

## 项目结构

```
src/
├── app/
│   ├── globals.css      # 全局样式
│   ├── layout.tsx       # 根布局
│   └── page.tsx         # 主页面
└── ...
```

## 设计理念

- **极简主义**: 采用黑白配色，去除冗余装饰
- **用户体验**: 直观的操作界面，清晰的信息层级
- **响应式设计**: 适配各种设备屏幕尺寸

## 部署

项目已配置为可以轻松部署到 Vercel 或其他支持 Next.js 的平台。
